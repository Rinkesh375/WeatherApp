## Daily Weather 

This is a simple weather forecast web application built with React.js that allows users to select a city and view its weather forecast for the next few days.

## live : - https://daily-weather-report.vercel.app/

## Tech Stack

- **Frontend**:
 - Technologies Used
 - React.js
 - Axios
 - OpenWeatherMap API
